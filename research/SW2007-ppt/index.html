问题 · Problem

随机奇异性

冲击数 < 可观测变量数 → 模型预测某些变量组合方差为零 → 似然函数退化，无法估计

GDP、消费、投资增长 + 劳动小时 + 实际工资 + 通胀 + 利率，恰好 7 个，避免随机奇异性，模型可估计

Eq.2–3 · Investment & Tobin's Q

投资的双向时间依赖

调整成本让投资峰值响应推迟，产生驼峰状 IRF

调整成本函数 S(·) 在稳态处 S=S′=0，S″=φ>0。投资对 Tobin's Q 的灵敏度 = 1/φ。

φ 越大 → 投资对 Q 越不敏感 → 响应越慢、越平滑

SW2007 后验均值 φ≈5.74，高于 CEE(2005)，意味着更强的投资调整摩擦
